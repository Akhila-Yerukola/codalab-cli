The goal of CodaLab Worksheets is to make researchers more efficient - check out this new short overview video to see what CodaLab is about.

Command-line interface

CodaLab comes with a powerful CLI, which you can now install/upgrade by simply running: `pip install -U codalab`.

Custom libraries / machines

Does your code fail because certain libraries aren't installed?  Learn how to build your own Docker images, which simply involves specifying a list of install commands.

Need GPUs?  CodaLab now has publicly available GPUs!  Just use the `--request-gpus` flag as shown in the CLI reference.  Also, you can use your own hardware by hooking up your own machines into CodaLab and have them only run your jobs. 

REST API

You can think about CodaLab as just a service for storing data and running commands.  We have exposed a REST API, which allows you to build new applications on top of CodaLab.  For example, we used the REST API to run a competition with 160 teams on SQuAD, a question answering dataset.

And as always, let us know if you have any questions or feedback.  Simply click on the new Help button on the bottom right of the page or submit a GitHub issue.  We hope you continue to enjoy using CodaLab!
