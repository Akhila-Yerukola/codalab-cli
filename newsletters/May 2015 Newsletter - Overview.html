Thank you for using CodaLab Worksheets! We wanted to give you an update on the newest features we’ve been rolling out. 

Installing custom packages

Want to use CodaLab with a specific library? You can install custom libraries on CodaLab by using Docker, which allows you to capture each of the environments your commands runs in. See the docs for more.

Using custom workers

Need custom hardware? You can use your own machines and plug them into CodaLab’s distributed worker system. Check out the docs for instructions on how.

REST API

Want to automate your workflows by integrating with CodaLab? You can check out the docs for our new REST API.

SQuAD Competition

Using CodaLab (via the REST API), we ran a competition with 160 teams using SQuAD, a question answering dataset with over 100,000 questions.  Read our blog post for details.

Pip install

We wanted to make it easier for you to install and upgrade your CodaLab CLI, so now it’s on PyPI! Install CodaLab’s CLI by using `pip install codalab-cli` and upgrade with  `pip install -U codalab-cli`.

Help Button

Have a question or comment on CodaLab? You can ask it directly on CodaLab by clicking the button on the bottom right of the page.

Overview Video

Want to share CodaLab with your colleagues? We created a video to show how CodaLab makes your research reproducible and more efficient.

And as always, let us know any questions or problems you’re having by opening an issue on our Github. We hope you continue to enjoy using CodaLab!
